Dollarss
========

An Instant Message Client by XMPP on iPhone

使用XMPP实现的iPhone上的聊天工具。

只完成了一小部分功能。

* 登录和注册;
* 获取联系人列表;
* 添加好友;
* 接受好友;
* 与好友聊天;
* 获取多人聊天房间列表;
* 加入房间;
* 房间内多人聊天;
* 修改个人状态;
